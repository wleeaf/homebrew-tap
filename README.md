# homebrew-tap

A [Homebrew](https://brew.sh) tap for [Projectum](https://github.com/wleeaf/projectum) on macOS.

```bash
brew install --cask wleeaf/tap/projectum
```

The build isn't notarized yet, so on first launch right-click the app and choose
**Open** (or System Settings → Privacy & Security → **Open Anyway**) to get past
Gatekeeper. Updates come with `brew upgrade --cask projectum`.
